# Healthcare-EDA
Healthcare Exploratory Data Analysis using Excel and Power BI

## Project Overview
This project focuses on performing exploratory data analysis on healthcare datasets
to identify trends, anomalies, and actionable insights that support patient care
improvement and resource optimization.

## Tools Used
- Advanced MS Excel
- Power BI

## Key Objectives
- Clean and transform raw healthcare data
- Perform exploratory data analysis (EDA)
- Create interactive dashboards and visual reports
- Deliver data-driven insights for stakeholders

## Project Structure
- `data/` – Raw dataset
- `excel_analysis/` – Excel-based analysis and pivot tables
- `powerbi_dashboard/` – Power BI dashboard (.pbix file)
- `screenshots/` – Dashboard visuals

## Key Insights
- Identified patterns in patient data and service utilization
- Highlighted high-impact areas for operational improvement
- Presented insights using clear and interactive dashboards
